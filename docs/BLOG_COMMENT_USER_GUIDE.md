## 🎯 Giriş

MarifetBul blog yorumları, topluluğumuzun fikirlerini paylaşması ve tartışmalar başlatması için tasarlanmış bir sistemdir. Bu kılavuz, yorum sistemini etkili bir şekilde kullanmanıza yardımcı olacaktır.

### Özellikler

✅ **Kolay Yorum Yapma** - Basit ve kullan

ıcı dostu arayüz  
✅ **Düzenleme** - Yorumlarınızı istediğiniz zaman düzenleyin  
✅ **Yanıt Verme** - Başkalarının yorumlarına yanıt verin  
✅ **Moderasyon** - Uygunsuz içerikleri bildirin  
✅ **Gerçek Zamanlı** - Yeni yorumları anında görün

## ✍️ Yorum Yapma

### Adım 1: Blog Yazısına Gidin

1. MarifetBul blog sayfasına gidin
2. Okumak istediğiniz yazıya tıklayın
3. Sayfa sonunda "Yorumlar" bölümünü bulun

### Adım 2: Yorum Formunu Doldurun

1. **Giriş Yapın**: Yorum yapmak için giriş yapmalısınız
2. **Yorum Yazın**: "Yorumunuzu yazın..." alanına tıklayın
3. **İçerik Girin**: Düşüncelerinizi yazın (maksimum 2000 karakter)

```
┌─────────────────────────────────────┐
│ Yorumunuzu yazın...                 │
│                                     │
│ Bu makale çok faydalıydı!          │
│ Özellikle örnekler konuyu          │
│ anlamama yardımcı oldu.             │
│                                     │
└─────────────────────────────────────┘
                            42/2000 karakter
                          [Yorum Yap]
```

### Adım 3: Yorumunuzu Gönderin

1. **Kontrol Edin**: Yazım hatalarını kontrol edin
2. **Gönder**: "Yorum Yap" butonuna tıklayın
3. **Onay Bekleyin**: Yorumunuz moderasyon için gönderilir

**Durum Göstergeleri:**

- 🟡 **Onay Bekliyor** - Yorum moderatör onayını bekliyor
- 🟢 **Onaylandı** - Yorum yayında ve herkes görebilir
- 🔴 **Reddedildi** - Yorum moderatör tarafından reddedildi

### İpuçları

✅ **Yapın:**

- Net ve öz yazın
- Saygılı olun
- Konuyla ilgili kalın
- Düşüncelerinizi destekleyin

❌ **Yapmayın:**

- Spam yapmayın
- Küfür kullanmayın
- Kişisel bilgi paylaşmayın
- Reklam yapmayın

## ✏️ Yorumları Düzenleme

Yorumunuzu gönderdikten sonra düzenleyebilirsiniz.

### Düzenleme Adımları

1. **Yorumunuzu Bulun**: Yaptığınız yorumu listede bulun
2. **Düzenle'ye Tıklayın**: Yorum kartındaki "Düzenle" butonuna tıklayın
3. **Değişiklik Yapın**: İçeriği düzenleyin
4. **Kaydedin**: Değişiklikleri kaydetmek için:
   - "Kaydet" butonuna tıklayın, VEYA
   - `Ctrl + Enter` kısayolunu kullanın

### Düzenleme Formu

```
┌─────────────────────────────────────┐
│ Ahmet Yılmaz • 5 dakika önce       │
│ ─────────────────────────────────── │
│ [Düzenleme modu aktif]              │
│                                     │
│ Güncelleme: Bu makale gerçekten     │
│ çok faydalıydı! Teşekkürler.       │
│                                     │
│                      58/2000 karakter│
│ [Kaydet] [İptal]                    │
└─────────────────────────────────────┘
```

### Klavye Kısayolları

- `Ctrl + Enter` - Değişiklikleri kaydet
- `Esc` - Düzenlemeyi iptal et

### Önemli Notlar

⚠️ **Dikkat:**

- Düzenlenmiş yorumlarda "Düzenlendi" etiketi görünür
- Onaylanmış yorumları düzenlerseniz tekrar moderasyona gidebilir
- Düzenleme geçmişi kayıt altına alınır

## 🗑️ Yorumları Silme

Artık istemediğiniz yorumlarınızı silebilirsiniz.

### Silme Adımları

1. **Yorumu Bulun**: Silmek istediğiniz yorumu bulun
2. **Sil'e Tıklayın**: "Sil" butonuna tıklayın
3. **Onaylayın**: Açılan diyalogda silme işlemini onaylayın

```
┌──────────────────────────────────┐
│   Yorumu Sil                     │
│                                  │
│   Bu yorumu silmek istediğinize  │
│   emin misiniz?                  │
│                                  │
│   Bu işlem geri alınamaz.       │
│                                  │
│   [İptal]  [Evet, Sil]          │
└──────────────────────────────────┘
```

### Önemli Bilgiler

⚠️ **Unutmayın:**

- Silinen yorumlar geri getirilemez
- Alt yanıtlar varsa onlar da silinir
- Silme işlemi hemen gerçekleşir
- İstatistikler güncellenir

## 💬 Yanıt Verme

Başkalarının yorumlarına yanıt vererek tartışmalara katılın.

### Yanıt Verme Adımları

1. **Yanıtla'ya Tıklayın**: Yanıtlamak istediğiniz yorumun "Yanıtla" butonuna tıklayın
2. **Yanıtınızı Yazın**: Açılan formda yanıtınızı yazın
3. **Gönderin**: "Yanıt Gönder" butonuna tıklayın

```
┌─────────────────────────────────────┐
│ Mehmet Demir • 1 saat önce          │
│ ─────────────────────────────────── │
│ Bu konuda daha fazla bilgi var mı?  │
│                                     │
│ [Yanıtla] [Bildir]                  │
│                                     │
│   ┌─────────────────────────────┐   │
│   │ @Mehmet Demir için yanıt    │   │
│   │                             │   │
│   │ Evet, şu kaynağa bakabilir- │   │
│   │ siniz: [link]               │   │
│   │                             │   │
│   │ [Yanıt Gönder] [İptal]      │   │
│   └─────────────────────────────┘   │
└─────────────────────────────────────┘
```

### Yanıt Hiyerarşisi

Yorumlar hiyerarşik yapıda gösterilir:

```
📝 Ana Yorum (Ahmet)
  ↳ 💬 Yanıt 1 (Mehmet)
    ↳ 💬 Yanıt 2 (Ayşe)
  ↳ 💬 Yanıt 3 (Can)
```

### İpuçları

- **@Mention**: Yanıt verirken kişi otomatik mention edilir
- **Bağlam**: Hangi yoruma yanıt verdiğiniz açıkça görünür
- **Bildirimler**: Yanıt aldığınızda bildirim alırsınız

## 🚩 Yorum Bildirme

Uygunsuz içerik gördüğünüzde moderatörlere bildirebilirsiniz.

### Bildirme Nedenleri

Bir yorumu bildirebileceğiniz nedenler:

1. **Spam veya Aldatıcı İçerik**
   - Reklamlar
   - Tekrarlayan mesajlar
   - Phishing linkleri

2. **Nefret Söylemi**
   - Ayrımcı içerik
   - Nefret dili
   - Taciz

3. **Taciz veya Zorbalık**
   - Kişisel saldırılar
   - Tehditler
   - Rahatsız edici davranış

4. **Şiddet İçeren İçerik**
   - Şiddet tehditleri
   - Zararlı içerik
   - Tehlikeli davranışlar

5. **Telif Hakkı İhlali**
   - Çalıntı içerik
   - İzinsiz kullanım
   - Marka ihlalleri

6. **Diğer**
   - Yukarıdakilere uymayan sorunlar

### Bildirme Adımları

1. **Bildir'e Tıklayın**: Yorumun "Bildir" butonuna tıklayın
2. **Neden Seçin**: Altı nedenden birini seçin
3. **Detay Ekleyin** (Opsiyonel): Ekstra açıklama yazın
4. **Gönderin**: "Raporu Gönder" butonuna tıklayın

```
┌──────────────────────────────────┐
│   Yorum Bildir                   │
│                                  │
│   Neden:                         │
│   ○ Spam veya aldatıcı içerik    │
│   ● Nefret söylemi               │
│   ○ Taciz veya zorbalık          │
│   ○ Şiddet içeren içerik         │
│   ○ Telif hakkı ihlali           │
│   ○ Diğer                        │
│                                  │
│   Detaylar (opsiyonel):          │
│   ┌──────────────────────────┐   │
│   │ Ayrımcı dil kullanılmış  │   │
│   └──────────────────────────┘   │
│                                  │
│   [İptal]  [Raporu Gönder]       │
└──────────────────────────────────┘
```

### Bildirme Sonrası

✅ **Ne Olur:**

- Raporunuz moderatörlere iletilir
- Yorum incelemeye alınır
- 24-48 saat içinde işlem yapılır
- Karar size bildirilir

⚠️ **Unutmayın:**

- Aynı yorumu birden fazla bildiremezsiniz
- Hatalı bildirimler hesabınızı etkileyebilir
- Acil durumlar için destek ekibine yazın

## ❓ Sık Sorulan Sorular

### Genel Sorular

**S: Yorum yapmak için giriş yapmam gerekiyor mu?**  
C: Evet, yorum yapmak için MarifetBul hesabınızla giriş yapmalısınız.

**S: Yorumlarım neden hemen görünmüyor?**  
C: Tüm yorumlar kalite kontrolü için moderasyona tabi tutulur. Genellikle 1-2 saat içinde onaylanır.

**S: Kaç karakter yorum yapabilirim?**  
C: Maksimum 2000 karakter yorum yapabilirsiniz.

**S: Yorumumu ne kadar süre içinde düzenleyebilirim?**  
C: Yorumunuzu istediğiniz zaman düzenleyebilirsiniz. Ancak onaylanmış yorumlar tekrar moderasyona girebilir.

### Moderasyon

**S: Yorumum neden reddedildi?**  
C: Reddedilen yorumlarda ret nedeni belirtilir. Topluluk kurallarına uyun ve tekrar deneyin.

**S: Başkasının yorumunu nasıl şikayet ederim?**  
C: Yorumun yanındaki "Bildir" butonuna tıklayın, nedeni seçin ve gönderin.

**S: Bildirimim ne zaman değerlendirilir?**  
C: Moderatörlerimiz tüm bildirimleri 24-48 saat içinde inceler.

### Teknik Sorunlar

**S: Yorumumu gönderirken hata alıyorum.**  
C: İnternet bağlantınızı kontrol edin. Sorun devam ederse destek ekibimize yazın.

**S: Düzenlediğim yorum kayboldu.**  
C: Tarayıcınızın cache'ini temizleyin ve sayfayı yenileyin.

**S: Mobilde kullanabilir miyim?**  
C: Evet, yorum sistemi tamamen mobil uyumludur.

### Güvenlik

**S: Yorumlarım güvende mi?**  
C: Evet, tüm yorumlar güvenli sunucularda saklanır ve şifrelenir.

**S: Yorumumu kim görebilir?**  
C: Onaylanan yorumlar tüm kullanıcılar tarafından görülebilir. Reddedilenler sadece size görünür.

**S: Hesabım kapatılırsa yorumlarım ne olur?**  
C: Yorumlarınız silinir ancak yanıtlar "Silinmiş Kullanıcı" olarak kalır.
